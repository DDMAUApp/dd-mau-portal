New-Hire Guide

⏱️ Day 1 — Get set up in 5 minutes

  1. Open the app in Safari (iPhone) or Chrome (Android/desktop) → ddmauapp.github.io
  2. Tap Share → Add to Home Screen (iPhone) or Install app (Android/desktop)
  3. Allow notifications when the app asks
  4. Tap Schedule → ⋯ → Set my availability
  5. Tap 🔒 Log out when you walk away from a shared device
📱
iPhone tip: Notifications only work if you install the app to your home screen. The browser tab won't get pushes — only the home-screen icon will.

💡 Tips veterans wish they'd known

  1. Install. The. App. Push notifications on iPhone REQUIRE the home-screen install. The browser tab won't work.
  2. Date-label everything you prep. Tap a recipe → Print prep label. Way faster than masking tape. Inspectors love it.
  3. Use the ✨ grammar fix in chat before sending if you're typing fast. It keeps your voice but cleans typos.
  4. Translate, translate, translate. The team is bilingual. Don't worry about which language to write in — every message can be translated with one tap.
  5. Set quiet hours in chat settings so you don't get pinged at midnight.
  6. 🔒 Log out on shared devices. PIN re-locks after 5 minutes idle, but be explicit.
  7. When in doubt, ask in chat. Someone always knows.
