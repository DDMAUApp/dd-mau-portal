## How to take a screenshot

**iPhone:** Press **Side button + Volume up** at the same time. AirDrop the
result to your laptop, OR upload through chat → save to disk → drop here.

**iPad:** Same, or press **Top button + Home button** on older iPads.

**Desktop:** macOS **Cmd+Shift+4** then drag the area you want. Saves to
the Desktop by default.

## Image guidelines

- **Format:** PNG (sharper for screenshots than JPG)
- **Aspect ratio:** Match the source device — phone shots stay portrait, desktop shots stay landscape
- **Max width:** ~1200px so the doc isn't massive
- **Crop the status bar** (the row with the time / battery / signal) — keeps
  the doc focused on the app, not your phone's clock

## After dropping in new screenshots

1. Open `docs/onboarding.html` in any browser
2. Your screenshots replace the dashed placeholder boxes automatically
3. **Cmd+P → Save as PDF** to get an updated PDF for new hires

If a screenshot doesn't appear, double-check the filename — it has to
match exactly (lowercase, hyphens, `.png` extension).
