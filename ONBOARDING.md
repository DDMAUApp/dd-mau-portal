## ⏱️ Day 1 — Get set up in 5 minutes

| # | What to do | Where |
|---|---|---|
| **1** | Open the app in **Safari** (iPhone) or **Chrome** (Android/desktop) | `ddmauapp.github.io` |
| **2** | Tap **Share → Add to Home Screen** (iPhone) or **Install app** (Android/desktop) | Browser menu |

| **4** | Tap **Schedule → ⋯ → Set my availability** | Inside the app |

> 💡 **iPhone tip:** Notifications **only work if you install the app to your home screen.** Tapping the same site in Safari doesn't get pushes — it has to be the home-screen icon.

## 🌟 Feature deep-dives

### 💬 Chat — How the team talks

> 📷 _Screenshot: `docs/images/03-chat-thread.png`_

| Action | Tap |
|---|---|
| Send a message | Type → tap ➤ |
| @-mention someone | Type `@` + their name |
| Translate a message | Tap the 🌐 chip below the message |
| Add a reaction | Long-press the message |
| Reply to a specific message | Swipe-right on it (or long-press → Reply) |
| Schedule a message for later | Composer → 📅 icon |
| **Polish spelling + grammar** | Composer → ✨ icon (AI does it in 1 sec) |
| Mute a channel | Settings → Notification settings → set channel to **Mute** |
| Edit your own message | Long-press → Edit (15-min window) |
| Mark someone out of stock (86) | Inside any chat → 🚫 button |

**Off-shift quiet hours:** Routine chat doesn't ping you between 10 pm and 6 am. **Mentions** and **86 alerts** always go through.

> 📷 _Screenshot: `docs/images/04-schedule-week.png`_

- 🟢 Today highlighted with a green stripe

| Action | How |
|---|---|

### 📋 Operations — The kitchen + ops home base

Sub-tabs at the top: **Tasks · Inventory · Prep · Sauce · 86**

> 📷 _Screenshot: `docs/images/05-operations-inventory.png`_

#### Tasks (Food Safety Checklists)
Opening / closing / mid-shift checklists. Tap to check off. Health-inspector ready.

#### Inventory
- Count items by **category**, **vendor**, or in **split mode**
- 🏷 **Label** button on the toolbar → quick date sticker on any container
- 🖨 **Print** button → free-form signs ("BROKEN", "RESERVED FOR PARTY OF 8")
- 🔍 **AI search** finds what plain search can't ("anything for pho", "spicy", "vegan")
- 🛒 **Cart**: items you've counted go here. Side-by-side vendor prices. **Order Mode** lets you check off items as they arrive.

### 🖨 Print Center — Word for your printer

> 📷 _Screenshot: `docs/images/08-print-center.png`_

Type anything. Print it. Sized for the 80mm linerless thermal printer at each kitchen.

**Quick templates (one tap):**
- 🚫 **Broken** / Do not use
- ⚠ **Quarantine** / Do not serve
- 🚨 **Allergen warning**
- 🔖 **Reserved** / Hold
- 🗓 **Custom date** label

**Controls:**
- Size: **Small / Normal / Large / Huge**
- **Bold** + alignment (left / center / right)
- 🗓 **Auto-stamp date+time** toggle
- ✍ **Auto-sign with my name** toggle
- **Copies 1-20** (batched into one print job)
- Last 6 messages you typed appear as one-tap recents

**How to get there:** 🖨 tile on the home screen, OR Operations → Inventory toolbar.

## 💡 Tips that veterans wish they'd known

1. **Install. The. App.** Push notifications on iPhone REQUIRE the home-screen install. The browser tab won't work.
2. **Date-label everything you prep.** Tap a recipe → Print prep label. Way faster than masking tape. Inspectors love it.
3. **Use the ✨ grammar fix in chat** before sending if you're typing fast. It keeps your voice but cleans typos.
4. **Translate, translate, translate.** The team is bilingual. Don't worry about which language to write in — every message can be translated with one tap.
5. **Set quiet hours** in chat settings so you don't get pinged at midnight.
6. **🔒 Log out on shared devices.** PIN re-locks after 5 minutes idle, but be explicit.
7. **When in doubt, ask in chat.** Someone always knows.
